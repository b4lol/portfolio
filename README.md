# b4lol

Source code for [b4.lol](https://b4.lol), a Hugo site with guides on digital
privacy and cybersecurity.

## Stack

- [Hugo](https://gohugo.io/)
- [WonderMod](https://github.com/Wonderfall/hugo-WonderMod) theme (submodule)

## Local development

```sh
git clone --recurse-submodules git@github.com:b4lol/portfolio.git
cd portfolio
hugo server -D
```
